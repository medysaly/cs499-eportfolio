## Sections

The following sections will be added throughout the capstone:

### 📹 Code Review *(Module 2)*
A video walkthrough of the original Unkommon codebase, identifying the areas planned for enhancement.

### 🛠 Enhancement One: Software Engineering and Design *(Module 3)*
Refactor the Unkommon backend into a layered service architecture with a CI/CD pipeline using GitHub Actions and OpenID Connect federation to AWS.

### ⚙️ Enhancement Two: Algorithms and Data Structures *(Module 4)*
Add a Trie-based intent classification layer that handles common FAQ queries before invoking AWS Bedrock, reducing latency and per-request cost.

### 🗄️ Enhancement Three: Databases *(Module 5)*
Redesign the DynamoDB data layer with a single-table design, Global Secondary Indexes for new access patterns, and a hot/cold separation pattern through DynamoDB Streams, S3, and Athena.

### 📝 Professional Self-Assessment *(Module 6)*
A reflection on the Computer Science program, the artifacts in this ePortfolio, and how they prepare me for an entry-level Cloud Engineer role.
